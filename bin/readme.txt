### Tips:

1. Use CTRL+SPACE to show up a auto-completer list.

2. Use source parser in Edit->Options menu to update your auto-completer
list to the latest version of your project. If you do not have a access
to source code, ask someone else for latest Functions.xml file.

3. Use function hints, it's useful! Simply hover mouse over a function
on the list and hint should appear if it's available.

4. Use Luna Helper! You can access single or multiple scripts a lot faster
by right clicking on file(s) and choosing "Edit with Luna Editor".

5. Take advantage of drag&drop functionality. Drop file on the field of text
to replace file, or drop it on tabs to add file to current set.

6. Right clicking on encoding type on status bar allows you to change it.
